功能特性

该有的都有

从检测到保存，全流程自动处理；录制、队列、历史一应俱全。

🎯
三路自动检测
同时拦截页面 XHR/Fetch 请求、监听网络响应 Content-Type、扫描 DOM 中的 <video> 元素，确保不遗漏任何流地址。
🔐
AES-128 解密
自动获取并缓存加密密钥，支持显式 IV 与序列号推导，使用 Web Crypto API 在浏览器内完成解密，无需额外工具。
🎬
多清晰度选择
解析 Master Playlist，按带宽从高到低排序，下载前可手动选择 1080p / 720p / 480p 等任意画质。
📦
fMP4 & BYTERANGE
完整支持 #EXT-X-MAP 初始化分片与 #EXT-X-BYTERANGE 字节范围请求，正确处理现代 HLS 格式并自动输出 .mp4。
⚡
实时速度 & ETA
4 秒滚动窗口计算瞬时下载速度，动态估算剩余时间，让你随时掌握下载进展，不再盲等。
✂️
分片范围选择
双滑块精确拖选起止分片，实时显示起止时间戳与选中时长，轻松截取视频中的任意片段。
🔴
直播流录制
自动识别直播流（无 #EXT-X-ENDLIST），一键切换录制模式，停止后自动合并为 .ts 文件保存到本地。
📋
批量下载队列
在弹窗中将多个流一键加入队列，后台 Service Worker 依次自动处理，无需手动守候每一个任务。
🕐
下载历史
自动记录最近 200 条下载记录，包含文件名、大小、分片数、域名，随时回溯已下载内容。
💾
低内存合并 & 持久设置
每 100 段批次合并 Blob 并释放 ArrayBuffer，避免 OOM；并发线程数通过 storage.sync 跨会话保存。

使用方式

四步完成下载

打开任何带有视频的页面，扩展自动完成检测与下载。

1
播放视频
正常播放网页视频，扩展后台自动捕获 M3U8 地址并更新 badge 计数
2
点击扩展图标
弹窗显示检测到的所有流，可直接下载、加入队列或手动粘贴地址
3
配置 & 选片段
选择画质、调节线程数，用双滑块选取分片范围，或切换录制模式
4
保存 & 查看历史
实时查看速度 / ETA / 日志，完成后自动记录到历史，支持多任务队列

安装

开始使用

目前需手动加载，Chrome Web Store 审核中。

手动安装（开发者模式）
1
前往 Releases 页面，下载最新的 chrome-mv3-*.zip
2
解压 zip 文件到本地任意目录
3
打开 Chrome，地址栏输入 chrome://extensions，右上角开启开发者模式
4
点击加载已解压的扩展程序，选择刚才解压的文件夹
